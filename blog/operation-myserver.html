2、使用 pm2 对服务器进程守护。

3、使用 docker 封装单个项目。

4、运行 docker。

具体操作步骤：

1、购买阿里云服务器

2、安装 nvm 并安装 node

1、docker 如何删除 none 镜像

如果你把 docker 当做容器用，就不需要。
