# 正则表达式之（先行断言，后行断言）

# 先行断言

语法：（?=pattern）

案例：匹配字符串x，但x后面必须是y

/x(?=y)/

# 先行否定断言

语法：（?!pattern）

案例：匹配字符串x，但x后面必须不是y

/x(?!y)/

# 后行断言

语法：（?<=pattern）

案例：匹配字符串x，但x前面必须是y

/(?<=y)x/

# 后行否定断言

语法：（?<!pattern）

案例：匹配字符串x，但x前面必须不是y

/(?<!y)x/
